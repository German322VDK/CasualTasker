Class Command

Represents an abstract base class that implements the System.Windows.Input.ICommand interface. Provides a simplified way to define command logic by handling CanExecuteChanged and delegating System.Windows.Input.ICommand.CanExecute(System.Object) and System.Windows.Input.ICommand.Execute(System.Object) to the derived class.
